# Fake News Detection System 📰❌✅

An AI-based Machine Learning project that classifies news articles as either **True** or **Fake** using Natural Language Processing (NLP) techniques.

## ✨ Features
* **Data Cleaning:** Removes HTML tags, punctuation, links, and special characters.
* **Text Vectorization:** Converts text into numbers using `TfidfVectorizer`.
* **High Accuracy:** Achieved **98.87% accuracy** using the `LogisticRegression` model.
* **Data Visualization:** Includes Count Plots and Confusion Matrix to analyze data distribution and AI performance.

## 📊 Visualizations & Results
Tumhari notebook ke andar ye dono graphs successfully generate ho chuke hain:
1. **Count Plot:** Shows the distribution of Fake vs True news in the dataset.
2. **Confusion Matrix:** Displays the exact number of correct and incorrect predictions made by the AI model.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, Scikit-Learn, Matplotlib, Seaborn, Regex (`re`), String
* **Environment:** Jupyter Notebook

1. Clone this repository or download the `Fake news detection.ipynb` file.
2. Install required libraries: `pip install pandas scikit-learn matplotlib seaborn`.
3. Open the file in Jupyter Notebook and run all cells.
4. Put any news line in the `testing_news` variable to check if it's True or Fake!
